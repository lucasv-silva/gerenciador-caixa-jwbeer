# 💰 Gerenciador de Caixa - JW Beer

Este projeto foi desenvolvido inteiramente por mim para resolver um problema real de organização financeira da minha adega.

## 🎓 Contexto Acadêmico
Sou aluno do curso de **Bacharelado em Tecnologia da Informação (BTI)** na **UFERSA** e este script em Python coloca em prática meus estudos de lógica de programação e manipulação de arquivos.

## 🚀 Funcionalidades
- Registro de entradas (vendas).
- Registro de saídas (gastos).
- Salvamento automático do saldo em um arquivo `.txt` no Linux.
- Interface simples e intuitiva via terminal.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3
- **Ambiente:** Linux (Xfce)
- **Editor:** VS Code
